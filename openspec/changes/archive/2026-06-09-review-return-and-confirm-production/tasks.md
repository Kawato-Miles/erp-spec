## 1. 印件狀態 enum 與 store 更新

- [x] 1.1 印件審稿維度 enum 新增 `confirmed_for_production`（已確認可製作）與 `pending_revision`（待改稿）兩個狀態值
- [x] 1.2 更新印件狀態轉換邏輯（store / utils）：新增 合格→已確認可製作、合格→待改稿、待改稿→等待審稿 三條轉換
- [x] 1.3 新增 B2C 自動確認邏輯：合格後若訂單為 B2C 類型，系統自動推進至「已確認可製作」
