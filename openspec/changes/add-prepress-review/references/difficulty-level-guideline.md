# 印件審稿難易度分級指引（草稿）

> 屬性：**非系統機制**，為業務在需求單 / 訂單印件 `difficulty_level` 欄位填寫時的參考。
> 對應 OpenSpec change: add-prepress-review tasks 1.2
> 最後更新：2026-04-19

## 使用原則

- **範圍**：整數 1 至 10，業務依印件的「稿件內容複雜度」與「審稿人員判讀難度」主觀判斷後填入
- **不等於印製難度**：本欄描述審稿（確認稿件是否達可送印狀態）的難度，與印製工序複雜度無關
- **繼承鏈**：需求單 → 訂單 → 工單 自動繼承；B2C 走 EC 商品主檔預設值

## 分級參考

| 等級 | 描述 | 常見印件類型 |
|------|------|------------|
| 1 | 標準模板、客戶使用品牌套版工具自助生成，稿件幾近固定格式 | EC 標準名片、EC 標準貼紙（B2C 模板） |
| 2 | 簡單單面設計，無特殊工藝，色彩 / 版型標準 | 純文字型手冊內頁、標準貼紙 |
| 3 | 單面或雙面平面設計，無多圖合成 | 一般傳單、簡單折頁 |
| 4 | 含少量圖像處理、出血與裁切需確認 | DM 摺頁、明信片 |
| 5 | 多元素版面設計，需審查色彩一致性與安全區 | 婚禮謝卡、品牌小卡、一般書籍封面 |
| 6 | 跨頁拼版、跨版型對齊、多素材整合 | 型錄、多頁手冊內頁 |
| 7 | 含特殊色或部分特殊工藝（打凸、簡易燙金） | 品牌手冊封面、禮盒內襯 |
| 8 | 多組特殊工藝（複合燙金、白墨、模切刀模線） | 精裝書衣、高規格禮盒 |
| 9 | 多層次特殊工藝同時出現（燙金 + 壓凸 + 模切 + 局部 UV） | 精品書衣、限量包裝 |
| 10 | 極高規格，任何工藝誤差皆影響品牌形象，需資深審稿人員確認 | 年度財報、品牌形象旗艦印品 |

## 與審稿人員能力對應

- 審稿主管於審稿人員管理介面設定每人 `max_difficulty_level`
- 系統自動分配時 SHALL 取「能力值最接近但 ≥ 印件難易度」者（能力最接近原則）
- 若所有審稿人員能力皆不足（candidate 為空），依 D13 破例派給當前能力最高者並記錄 ActivityLog

## 填寫建議

1. **先看印件類型**：若屬 B2C 商品主檔，直接套用商品主檔的預設值
2. **再看工藝**：有特殊工藝至少 +2 級；有多組特殊工藝（例燙金 + 壓凸）至少 7 級起跳
3. **再看版面**：多圖、多文字、跨頁版型對齊至少 +1 級
4. **最後看品牌要求**：品牌對色彩或細節要求嚴格的 B2B 客戶，可再 +1 級
5. **例外原則**：若稿件內容為客戶首次合作且品牌識別嚴格，難易度 SHALL ≥ 7（確保由資深審稿人員承接）

## 後續治理

- 業務團隊每季檢討難易度分級結果與實際審稿時長 / 退件率的相關性
- 若某一等級（如「5」）出現退件率顯著偏高，審稿主管可與業務討論調整分級判準
- 本指引為草稿版本，依實際運作狀況持續修訂
