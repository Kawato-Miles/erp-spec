# 訂單（Order）

> ERP 核心實體。**三種訂單類型**：一般訂單（需求單成交）/ 諮詢訂單（諮詢結尾）/ EC 線上訂單。

## 核心欄位

| 欄位 | 說明 |
|------|------|
| `id` / `order_no` | 系統 ID / 業務看見的編號 |
| `status` | 狀態（見 [[訂單狀態]]，線下 / 線上 / 諮詢路徑不同）|
| `order_type` | `一般` / `諮詢` / `EC` |
| `customer_id` | 客戶 |
| `sales_id` | 負責業務 |
| `billing_company_id` | 帳務公司 |
| `payment_terms_note` | 收款條件備註（從需求單帶入）|
| `case_name` | 案件名稱 |
| `expected_delivery_date` | 預定交期 |
| `signed_at` | 報價回簽時間 |
| `linked_quote_request_id` | 來源需求單 → [[需求單]] |
| `linked_consultation_request_id` | 來源諮詢單 → [[諮詢單]] |
| `completed_at` | 訂單完成時間 |

## 關鍵關聯

- 包含 `OrderPrintItem[]`：訂單下的印件 → [[印件]]
- 每筆 OrderPrintItem 包含 `WorkOrder[]` → [[工單]]
- 包含 `OrderAdjustment[]`：訂單異動單
- 包含 `Invoice[]` + `SalesAllowance[]`：發票 + 折讓單
- 包含 `Payment[]` + `PaymentPlan[]`：付款紀錄 + 分期付款計畫
- 包含 `ShippingRecord[]` → [[出貨單]]
- 線上訂單關聯 EC payment
- 已完成後可有 `AfterSalesTicket[]` → [[售後服務]]
