# 生產任務（ProductionTask）

> 工單內的**單一生產任務**（如印刷、裁切、燙金）。每個生產任務有負責的廠商。

## 核心欄位

| 欄位 | 說明 |
|------|------|
| `id` | 系統 ID |
| `work_order_id` | 所屬工單 → [[工單]] |
| `task_id` | 所屬任務群組 → [[任務]] |
| `bom_type` | `material` / `process` / `binding` |
| `material_spec_id` / `process_id` / `binding_id` | BOM 三類互斥 FK |
| `production_line_id` | 產線 |
| `factory_type` | 工廠類型（自有 / 加工 / 外包 / 中國廠商）|
| `quantity_per_work_order` | **完成 1 份工單需要多少份此生產任務**（見 [[數量換算規則]]）|
| `affects_product` | 是否計入工單完成度（TRUE / FALSE）|
| `status` | 狀態（見 [[生產任務狀態]]）|
| `estimated_duration_days` | 預估工期 |
| `pt_scheduled_date` / `planned_end_date` | 排程開始 / 結束日 |
| `assigned_operator` | 執行人（師傅 / 廠商）|

## 關鍵關聯

- 屬於 `WorkOrder` → [[工單]]
- 屬於 `Task`（同工廠分組）→ [[任務]]
- 包含 `WorkReport[]`：報工紀錄
- 包含 `QCRecord[]` → [[QC]]
- 引用 BOM：material_spec / process / binding 三互斥 FK
- 若有異動則涉及 `WorkOrderModification.pt_snapshots`

## 工廠類型決定狀態路徑

| 工廠類型 | 狀態路徑 |
|---------|---------|
| 自有工廠 | 待處理 → 製作中 → 已完成 |
| 加工廠 | 待處理 → 製作中 → 已完成 |
| 外包廠 | 待處理 → 製作中 → 運送中 → 已完成 |
| 中國廠商 | 待處理 → 製作中 → 已送集運商 → 運送中 → 已完成 |

詳見 [[生產任務狀態]]。

## TransferTicket（轉交單）

印件級跨站點運送憑證（add-production-task-transfer change 引入）：

- 透過 `printItemId` FK 關聯印件，一印件可有多張以支援分批轉交
- 每張單可含 `lines[]` 明細抽取同印件下多個生產任務的產出，但**不可跨印件**
- 狀態機：運送中 / 已送達 / 已作廢

詳見 [[glossary-erp#生產排程術語]] § 轉交單。

## 終態與作廢

- **已作廢**：待處理時可作廢
- **報廢**：製作中時可報廢
- 生產任務作廢時，**關聯的 QC 單自動作廢**（見 [[QC]]）
