# 生產任務狀態（ProductionTaskStatus）

> [[生產任務]] 的狀態機。**工廠類型決定狀態路徑**（4 種路徑）。

## 工廠類型決定狀態路徑

### 1. 自有工廠路徑

```
待處理 → 製作中 → 已完成
```

### 2. 加工廠路徑

```
待處理 → 製作中 → 已完成
```

### 3. 外包廠路徑

```
待處理 → 製作中 → 運送中 → 已完成
```

### 4. 中國廠商路徑

```
待處理 → 製作中 → 已送集運商 → 運送中 → 已完成
```

## 其他終態

- **已作廢**：待處理時可作廢
- **報廢**：製作中時可報廢

## 主要轉移觸發

| From → To | 觸發 | 推進方式 |
|-----------|------|---------|
| 待處理 → 製作中 | **首次報工**（[[師傅]]或供應商） | 自動推進上層 |
| 製作中 → 已完成（自有 / 加工）| 完成製作並上報 | 手動 |
| 製作中 → 運送中（外包）| 完成製作並上報 | 手動 |
| 製作中 → 已送集運商（中國）| 完成製作並上報 | 手動 |
| 運送中 → 已完成 | 貨物到達 | 手動 |
| 已送集運商 → 運送中 | 集運商出貨 | 手動 |
| 待處理 → 已作廢 | 印務作廢 | 手動 |
| 製作中 → 報廢 | 異動時報廢 | 手動 |

## 自動向上推進

```
生產任務首次報工 → 自動推進「製作中」
  → 任務狀態自動「製作中」
  → 工單狀態自動「製作中」
  → 印件狀態自動「部分工單製作中」/「製作中」
  → 訂單狀態自動「製作中」
```

詳見 [[任務狀態]] / [[工單狀態]] / [[印件狀態]] / [[訂單狀態]]。
