# XM-007：降級為次級品出貨給其他客戶的處置流程

## 問題描述

reclassify-qc NCR + Disposition 三選一（rework / use_as_is / scrap）僅涵蓋「廠內驗收當下處置」。**未涵蓋情境**：

- 印件不良但仍可使用 → 不退原客戶（rework 過 / 已交付）→ 降級為次級品出貨給其他願接受次級品的客戶

現靠業務口頭協調，無系統紀錄，無法追溯次級品流向、無法統計次級品銷售比例。

## 涉及範圍

production-task / order-management / cross-module（廠內驗收 + 業務銷售 + 出貨）

## 候選方案

C5 或之後 change 處理（範圍另議）。需要設計：
1. 次級品標記機制（NCR 新增 disposition = `downgrade`？或獨立實體？）
2. 次級品庫存管理（與正級品分開計帳）
3. 次級品出貨流程與發票

詳見 [reclassify-qc design.md § Decisions 10 § 客訴退貨 / 降級出貨邊界](../../../../openspec/changes/archive/2026-05-20-reclassify-qc-and-add-inspection/design.md)。
