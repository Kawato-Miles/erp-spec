## 一、報價流程概要

```
業務建需求單 → 印件難易度 / 規格填寫 → 待評估成本 → 印務主管成本評估 → 已評估成本 → 業務 / 印務報價 → 議價 → 成交 / 流失
```

## 二、需求單狀態機

| 狀態 | 說明 |
|------|------|
| 需求確認中 | 業務蒐集客戶需求 |
| 待評估成本 | 業務送出，等待印務主管評估 |
| 已評估成本 | 印務主管完成評估，業務可報價 |
| 議價中 | 業務與客戶談價 |
| 成交 | 進入訂單階段 |
| 流失 | 客戶不接受報價 |

> 業務獨立完成，**沒有業務主管 gate**。業務主管審核發生在訂單階段（訂單建立後 → 報價待回簽前），見 [[訂單狀態]]。
