# 訂單管理人

> [!info] OpenSpec spec 狀態：**缺漏**
> `openspec/specs/user-roles/spec.md` 中無此角色，需後續評估是否補建。詳見 [[_alignment-report]]。

## 基本資料

- **平台**：中台
- **利害關係程度**：高
- **參與階段**：評估 / 訂購 / 審稿 / 打樣印製 / 出貨（全程）
- **任務操作**：R/W
- **報價單 / 訂單**：R/W
- **工單**：R/W
- **需求單**：X

## 主要工作職責

追蹤與管理公司訂單進度；確認印務分配狀況

## 關切點

- 工單分配效率
- 整體進度可見
- 合批派工功能減少重複整理

## 預期阻力

- 現有手動整理派工表的習慣需改變
- 系統合批派工的合併邏輯是否符合實際操作需確認（OQ XM-003）
