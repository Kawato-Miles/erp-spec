# 供應商報價規則

> 本卡承載外包廠商報價提交與審核的業務規則完整定義。屬生產與物流領域。

## 營運背景

印刷廠部分工序外包給供應商（外包廠），生管把生產任務分派出去後，供應商要報價、生管要審核。實務上報價和生產常常並行——供應商已經開工了但報價還沒回來，因此兩者不能互相卡住。

## 規則與營運動機

### 報價審核三階段

| 項目 | 內容 |
|------|------|
| 意圖 | 供應商報價有明確的提交與審核流程，生管有權確認或退回 |
| 規則 | 流程為：供應商報價（待報價 → 已報價） → 生管審核 → 確認（已確認）或退回（已退回、供應商重新報價）。退回時生管填寫退回原因，供應商收到通知後重新報價 |
| 營運動機 | 外包報價品質參差，生管需要一個把關環節。退回機制讓不合理的報價有修正空間，而非只能接受或另找廠商 |

### 報價與生產可並行

| 項目 | 內容 |
|------|------|
| 意圖 | 報價流程不卡住生產進度 |
| 規則 | 供應商尚未報價但生產任務已可開工時，系統允許供應商先開始生產（報工）再補報價。報價流程與生產狀態機獨立運作，互不阻擋 |
| 營運動機 | 印刷業的交期壓力大，等報價核定才開工會延誤交期。讓生產先跑、報價後補，是台灣印刷業的常態做法 |

## 具體例子

例子 1（報價正常流程，正例）：生管把 500 張海報的裝訂任務分派給外包廠 A，報價狀態為「待報價」。外包廠 A 在供應商平台提交報價 3000 元，狀態變為「已報價」。生管審核後確認，狀態變「已確認」。

例子 2（報價退回，正例）：外包廠 B 報價 8000 元，生管認為超出行情，退回並附理由「同規格市場價約 5000」。外包廠 B 收到通知後重新報價 5500 元，生管確認。

例子 3（報價與生產並行，邊界正例）：外包廠 C 的報價還在「待報價」，但交期緊迫，外包廠 C 先開工報工了 200 張。報價流程獨立進行，不影響已報工的進度。
