# PI-006 免審印件工單派工的加工檔與縮圖來源

## 問題描述

工單派工時，工作包帶入「合格輪次的加工檔（審稿後上傳檔案）＋縮圖」。依 [[稿件管理規則]]，這兩個檔案是**審稿人員審查後才產出**的。

但免審印件跳過整個審稿流程（[[免審決策樹]] / 稿件管理規則例外明載免審稿不適用本規則），沒有審稿人員產出的加工檔與縮圖——免審印件的工單派工時，工作包的加工檔／縮圖從何而來，wiki 全未定義。這是現有免審就埋著、一直未被點出的缺口，大貨參考打樣只是讓它浮現。

## 待解答

- [ ] 大貨參考打樣的加工檔／縮圖，走「引用打樣印件合格輪次那一份」還是「複製一份進大貨印件」？（複製：工單鎖檔機制不動、大貨印件稿件維度自包含；引用：單一來源、但鎖檔要能跨印件取檔）
- [ ] 名片補印類一般免審印件（與打樣無關）的加工檔／縮圖從哪來：客戶提供？沿用上一次相同品項訂單的檔案？系統如何取得並掛進工單？
- [ ] 兩類免審（打樣後參考、重複品項）的加工檔來源是否該用同一機制，還是各自獨立？

## 範圍切分

- 大貨參考打樣這個 case 有明確來源（打樣印件合格輪次），可優先解。
- 名片補印類一般免審的加工檔來源是另一條現況，須另與業務／工廠確認實務，不在本次釐清臆測。
