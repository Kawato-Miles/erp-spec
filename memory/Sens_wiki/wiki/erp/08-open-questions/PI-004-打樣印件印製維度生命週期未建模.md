# PI-004：打樣印件印製維度的生命週期，印件狀態機未建模

## 問題描述

[[印件狀態]] 的印製維度（等待中 → 工單已交付 → 製作中 → 製作完成 → 出貨中 → 已送達）是為**大貨印件**設計的：以出貨單驅動、「已送達」是「所有出貨單皆送達且累計送達數量等於印件數量」的終態、「等待中」是單向初始態無回頭轉換。

但**打樣印件**借用同一條印製維度，其生命週期與大貨印件不同，狀態機未涵蓋兩個關鍵語意：

1. **樣品送樣不是出貨單**：打樣印件做完後是「寄送樣品給客戶評審」，不建出貨單；用印製維度「已送達」承載「樣品送到客戶手上」，是把狀態名套到狀態機未定義的觸發來源。
2. **重打樣（NG-製程問題）是終態回頭**：打樣 NG-製程問題時於同一打樣印件下重做一輪打樣，等於印製維度從一個近終態回到「重新製作」——但印件狀態機的「等待中」是單向初始態、近終態無回頭邊，此轉換在狀態機中不存在。
