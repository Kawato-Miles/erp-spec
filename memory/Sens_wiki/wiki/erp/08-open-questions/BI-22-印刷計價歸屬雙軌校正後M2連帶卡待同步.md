# BI-22 印刷計價歸屬雙軌校正後 M2 連帶卡待同步

## 問題

M2 裁決「印刷計價歸工序、設備只填產能參數」已於 2026-07-23 依 erp repo 現況校正推翻為**雙軌計價**（印刷計價掛設備、後加工計價掛工序）。[[設備]] 與 [[工序主檔]] 兩張卡已改為雙軌正本，但 M2 舊論述仍散在四張連帶卡未同步，造成全庫計價歸屬暫時不一致。待同步下列四卡。

## 背景

- 現況雙軌來源：erp repo 設備 service（`equipment` 承載整套印刷計價：平版／數位／大圖三型別、令數／千車階梯、色數與特殊色倍率）＋工序 service（`process` 八種 `pricing_method` 承載後加工計價）。
- Miles 2026-07-23 裁決：以現況雙軌為正本、推翻 M2；四主檔（設備／工序／材料／裝訂）先改，M2 連帶卡「另議」。
- 本 OQ 承接「另議」的連帶同步，非待裁決問題（方向已定），追蹤同步進度與暫時矛盾狀態。

## 待同步的連帶卡

- [[2026-06-13-生產模組架構設計]]：M2 裁決正本所在（計價歸屬論述需改雙軌）
- [[BOM結構]]：計價公式歸屬規則正本（印刷計價公式歸屬需由工序改設備）
- [[成本差異]]：含 M2 計價歸屬相關論述
- [[2026-06-15-生產階段現況校正與as-is-tobe對齊]]：含 M2 相關對齊論述

## 影響範圍
