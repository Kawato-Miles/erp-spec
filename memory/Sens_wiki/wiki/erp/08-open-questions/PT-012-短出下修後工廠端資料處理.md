# PT-012：短出下修後工廠端資料處理

## 問題描述

品檢缺口決定不補做時，收尾順序已拍板（2026-07-22）：業務先在訂單下修數量、印務再處理工廠端資料。但「印務處理工廠端資料」的系統功能未定：業務下修訂單後，工廠端的工單／生產任務要怎麼收——改數量還是改狀態、介面長什麼樣、與訂單下修怎麼連動。

## 待解答

- [ ] 工單／生產任務是改數量（目標數量下修對齊）還是改狀態（如標記提前收尾）？
- [ ] 印務處理工廠端資料的介面與操作入口為何？
- [ ] 訂單下修與工廠端資料處理之間是否要系統連動（下修後自動提示印務待處理清單）？
