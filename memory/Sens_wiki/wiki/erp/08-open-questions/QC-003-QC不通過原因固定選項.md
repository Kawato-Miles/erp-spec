# QC-003：QC 不通過原因（固定選項）欄位與選項清單

## 問題描述

生產階段 to-be 流程草稿原寫「品檢記錄不通過時帶固定選項原因」，但 [[QC]] 實體卡目前僅有通過數量 / 不通過數量，**無「不通過原因」欄位**（對抗式查核抓出草稿越界）。

此欄位的價值是「品質數據化入口」：有固定選項原因後，才能統計各主因佔比、各工序 / 各外包不良率、色差 vs 刮傷等分佈，支撐良率細分與外包品質管理，呼應痛點「無法查產能、耗損率」。成本低（QC 記錄多一個下拉欄位）。

## 涉及範圍

- 模組：品檢、生產任務
- 相關卡：[[QC]]（欄位正本）、[[QC 狀態]]、[[生產任務]]、[[品檢人員]]
- 影響範圍：
  - [[QC]] 實體卡新增「不通過原因」欄位（欄位正本歸 wiki）
  - 未來良率 / 不良率 KPI 細分（[[生產績效指標]]）的分組維度來源
  - 與 [[PT-003-NCR Rework 具體實現|PT-003]]（不良品處置 NCR/Disposition）的銜接（原因 → 處置）

## 部分拍板

- 2026-06-15 Miles 拍板：QC「不通過原因」欄位**要做**，並以業界初版選項先上、後續再改。
- 仍 open：固定選項清單的最終內容（待現場品檢 / 印務收斂）、欄位是單選或可複選、是否每筆不通過數量需各標一個主因。

## 待解答

- [ ] 固定選項清單最終內容（現場品檢 / 印務確認、增刪初版選項）
- [ ] 欄位採單選（一筆不通過數標單一主因）或可複選
- [ ] 不通過數量是否需依原因拆分計數（如「30 件不通過 = 色差 20 + 刮傷 10」），還是僅標主因

## 候選方案（初版業界選項，待現場收斂）

依不良發生環節分組，作為下拉初版（現場可增刪）：

| 分組 | 固定選項（初版） |
|------|----------------|
| 印刷面 | 色差／偏色、套印不準、糊版／印紋不清、漏印／墨色不均、墨點／油污 |
| 表面加工 | 上光（亮膜／霧膜）起泡或刮花、燙金／打凸位置偏或脫落、局部 UV 偏位 |
| 裁切／成型 | 裁切尺寸不符、裁切／摺線偏移、開孔／刀模位置偏（壓克力、立體件） |
| 裝訂 | 騎馬釘／膠裝／穿線不良、書背歪、頁序錯 |
| 材料 | 紙張／板材瑕疵（紙痕、皺摺、破損）、厚度不符、邊緣崩裂 |
| 規格／數量 | 做錯版本或規格、材質錯誤、數量不足／缺件 |

初版選項先寫入 [[QC]] 欄位設計即可運轉；最終清單待併入生產／出貨現場訪談批次與品檢確認後收斂（external）。
