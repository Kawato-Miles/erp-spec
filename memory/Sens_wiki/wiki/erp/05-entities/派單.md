# 派單（OutsourceTicket）

> 一句話定位：把指定稿件（印件）委外給外部供應商製作的外發委外單，屬生產執行領域。現況實作以**稿件為單位**：一筆派單＝一筆外發稿件，指派資訊（供應商、期望交期、指派時間）掛在稿件上。它記外部廠商從承接、發稿、製作、出貨到集運回台的當地進程，以及這筆稿件的委外金流（報價、運費、關稅分攤與毛利率）。外發委外的決策政策（哪種廠商才產生派單）正本在 [[印件生產流程]]，外部進程狀態正本在 [[派單狀態]]，本卡只引用不重述。

## 這張卡要回答的問題

- 工單已經能代表一條製作活了，為什麼還要另開一張派單？→ 看「營運背景」（派單記的是工單代表不了的外部廠商當地進程與委外金流）。
- 哪些製作會產生派單、哪些不會？→ 看「營運背景」（廠商類型路由：外包廠／中國廠才產生派單；場內與加工廠不外發、不需派單）。
- 一張派單上業務看得到哪些資料？→ 看「欄位（業務可見）」與「運費紀錄（子表）」。
- 委外的錢怎麼算？→ 看「委外金流」欄位群（供應商報價、匯率、分攤運費、固定運費、分攤關稅、毛利率）。
- 派單跟工單、印件、生產任務、供應商怎麼牽連？回台物流去哪查？→ 看「關鍵關聯」（回台物流見 [[貨運單]]）。
- 派單有哪些外部進程狀態？→ 看「相關狀態機」（[[派單狀態]]，13 值大陸處理狀態鏈）。

## 營運背景

派單是一張**獨立的外發委外單**，跟工單是兩回事。工單代表「這件活要做」，派單代表「這件活外發給某家外部供應商做、現在做到哪了、委外花了多少錢」。當製作要交給外部廠商時，派單承載工單代表不了的東西——外部廠商的承接、發稿、製作、分批出貨、集運、回台這一整段當地進程，加上供應商報價與運費關稅的成本歸集。

不是每件活都產生派單，依**廠商類型路由**：只有外發給外包廠或中國廠的製作才產生派單；場內（自有產線）與加工廠的製作不外發，不需要派單。中國線的派單已在現況平台實作（供應商如雅那印刷、第一／第二／第四印刷）；台灣外包廠的派單概念相同，但尚未實作。

派單記的進程與生產任務的內部製作狀態分離：生產任務是內部單位、粗顆粒狀態，外發期間的外部細節（整批／分批製作、分批出貨、海外直發等）都歸派單的大陸處理狀態鏈追蹤。打樣、大貨、補印是工單製作類型（工單屬性），不是進程狀態——重打與補印以新工單（新派單）另起一條進程，見 [[派單狀態]]。

## 欄位（業務可見）

> 欄位忠於現況派單管理頁（中國線已實作）。台灣外包廠尚未實作，屆時回補。

### 委外金流

> 人民幣金額以稿件鎖定匯率換算新台幣；匯率為全域設定（人民幣對台幣，只追加新值不覆蓋舊值），稿件於派發時鎖定當下匯率。

| 介面欄位 | 說明 | 欄位形態 | 來源 | 可否修改 |
|---------|------|---------|------|---------|
| 數量／單價／總價 | 此稿件的商品數量與金額（新台幣） | 數字／金額 | 計價帶入 | 唯讀 |
| 供應商報價 | 供應商對此稿件的報價，以單價或總價擇一方式調整 | 金額 | 供應商報價、管理員可調整 | 可調（單價或總價二選一） |
| 匯率 | 此稿件鎖定的人民幣對台幣匯率 | 數字 | 派發時鎖定全域匯率 | 唯讀 |
| 分攤運費 | 依重量佔比攤到此稿件的集運運費（見 [[貨運單]] 分攤） | 金額 | 系統計算 | 唯讀 |
| 固定運費 | 此稿件的固定運費（不走重量分攤的運費部分） | 金額 | 系統計算 | 唯讀 |
| 分攤關稅 | 依重量佔比攤到此稿件的關稅（見 [[貨運單]] 關稅） | 金額 | 系統計算 | 唯讀 |
| 毛利率 | 此稿件的毛利率（百分比） | 數字 | 系統計算 | 唯讀 |

### 運費紀錄（子表，每條＝一筆寄送運費）

> 稿件層的運費登錄，台灣線與中國線的承運商選項不同（中國線：順豐／集運-巧巧狼／集運-台集運／其他自填；台灣線：順豐／新竹／集運）。人民幣金額依稿件鎖定匯率換算台幣。

| 欄位 | 說明 |
|------|------|
| 運費類型 | 台灣或中國寄送 |
| 貨運廠商 | 承運的物流商名稱 |
| 寄件單號 | 物流追蹤單號 |
| 重量 | 這筆寄送的重量 |
| 金額與幣別 | 運費金額（台幣或人民幣），人民幣依鎖定匯率換算台幣 |
| 寄出日期 | 寄出那一天 |
| 備註 | 補充說明 |

### 範圍外（存在但屬實作規格，不列入本欄位表）

- 系統識別碼、外鍵、時間戳記等純技術欄位。
- 面板上「工單狀態」（審稿合格／製作中（大陸）／運送中／已完成／補印中／工單已作廢）為 [[工單]] 本身的狀態，正本在工單與 [[工單狀態]]，派單只是顯示對應工單的當下狀態。
- 批量編輯（選多筆稿件一次更新同一欄位）為操作功能，非欄位。

## 關鍵關聯

- 綁定一張 [[印件]]（派發的是指定稿件；現況實作以稿件為派單單位）。
- 對應一張 [[工單]]（派單溯源的工單，1 派單對 1 工單；工單製作類型決定這條進程承載打樣、大貨或補印）。
- 對應內部的 [[生產任務]]：外發期間外部進程歸派單追蹤，生產任務維持內部粗顆粒狀態；回台物流的運費與關稅按重量佔比分攤回稿件與原本的生產任務（成本回生產任務，詳見 [[貨運單]]）。
- 指派一家外部供應商（[[外包廠商]] 或 [[中國廠商]]，外發才產生派單）。
- 回台跨境物流見 [[貨運單]]（中國→台灣的承運、申報運費、收貨點收、重量差異、運費關稅分攤、關稅單據）；一張貨運單可關聯多張派單（集運合併）。
- 匯率為全域設定、稿件派發時鎖定，運費與關稅換算皆用鎖定匯率。

## 相關狀態機
