# 印務主管

## 基本資料

- **平台**：中台
- **利害關係程度**：高
- **參與階段**：評估 / 打樣印製
- **任務操作**：X
- **報價單 / 訂單**：X
- **工單**：R/W
- **需求單**：R/W

## 主要工作職責

售前製程與成本評估；印製工單製程審核、BOM 表管理；建立工單草稿；分配工單給印務；追蹤工單執行狀況

## 關切點

- 工單審核有系統支援
- 異動有留存記錄
- 可依角色篩選追蹤操作歷程

## 預期阻力

審核流程從口頭 / Slack 改為系統操作，初期可能因操作不熟悉而增加確認時間
