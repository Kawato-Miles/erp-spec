## 月結對帳節奏

> 2026-05-28 Miles 對談補充（來源：BI-13 第一輪稽核對談）。

- **結帳週期**：每月**月底結帳**
- **對帳頻率**：通常**每月或每兩週**對帳一次
- **對帳時動作**：發現「發票已開立但未收款」時，**提醒業務追款**（見 [[業務#款項追款流程]]）
- **三方對帳來源**：**銀行帳款資料 + 發票 + 訂單資料**三方比對（對應 [[付款發票邏輯#六、三方對帳]]：應收 = 發票淨額 = 收款淨額）
- **退款異動單核對**：對帳時依「已執行」的退款訂單異動單金額核對退款款項——已執行的退款異動單必對應一筆已完成的退款款項（補收正項異動單不涉及退款款項，不在此核對範圍）。會計不參與訂單異動單的建立 / 審核 / 執行，僅在對帳時核對其金額流向（規則正本見 [[訂單異動規則]]）
- **對 ORD-001 的證據**：本段證實會計**參與訂購 / 出貨 / 月結全程對帳**（涉及錢的全程），非 frontmatter「僅審稿階段」標記 → 強化 [[ORD-001-會計階段標記是否錯誤]] 應修正（待 Miles 正式拍板後改 frontmatter 參與階段）
