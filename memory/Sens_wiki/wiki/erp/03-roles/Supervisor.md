# Supervisor

> [!info] OpenSpec spec 狀態：**缺漏**
> `openspec/specs/user-roles/spec.md` 中無此角色，需後續評估是否補建。詳見 [[_alignment-report]]。

## 基本資料

- **平台**：中台
- **利害關係程度**：高
- **參與階段**：評估 / 訂購 / 審稿 / 打樣印製 / 出貨（全程）
- **任務操作**：R/W
- **報價單 / 訂單**：R/W
- **工單**：R/W
- **需求單**：R/W

## 主要工作職責

Dashboard、Forecast、KPI 追蹤

## 關切點

- 產能透明度
- 訂單不掉單
- 生產進度隨時可查

## 預期阻力

- 導入初期資料不完整，產能可見度未達預期時的耐心
- 對系統建置週期與 Phase 切換時間點的期待管理

## 備註

不直接編輯資料
