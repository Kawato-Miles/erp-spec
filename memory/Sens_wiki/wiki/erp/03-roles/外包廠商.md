# 外包廠商

## 基本資料

- **平台**：工廠平台
- **利害關係程度**：低（外部）
- **參與階段**：打樣印製 / 出貨
- **任務操作**：R/W
- **報價單 / 訂單**：X
- **工單**：X
- **需求單**：X

## 主要工作職責

執行生產任務並完成報工；確認成品如期完成

## 關切點

- 工單指示清楚
- 進度回覆方式直覺
- 系統操作不複雜

## 預期阻力

- 外部合作夥伴，非所有人都願意使用系統
- ERP 尚未支援時以紙本傳真作為過渡方案

## 備註

- 僅可編輯任務進度回覆
- 不可更改任務內容
- 系統未支援時採紙本傳真過渡

## 與中國廠商的差異

兩者都是外部廠商，主要差異：
- **中國廠商**：在中國供應商平台，**可報價**
- **外包廠商**（本卡）：在工廠平台，**不報價**（僅執行 + 確認成品）+ 參與出貨階段
