# 印刷業產業背景知識

## 產業概述

印刷業是高度客製化、流程複雜的製造業，同時也正在進行數位化轉型。主要分為商業印刷（型錄、DM、名片）、包裝印刷（紙箱、標籤）、出版印刷（書籍、雜誌）與特殊印刷（布料、塑料、金屬）。

---

## 核心價值鏈

```
客戶下單 → 稿件提交 → Preflight 檢查 → 拼版排程 → 印刷生產 → 印後加工 → 裁切 → 出貨
```

每個環節都有其術語、流程節點與常見瓶頸。

---

## 主要業務模式

| 模式 | 說明 | 特性 |
|------|------|------|
| B2B 傳統印刷廠 | 接受企業客戶長期訂單 | 訂單量大、客製化高、業務關係導向 |
| 線上印刷平台 | 標準化產品、自助下單 | 高自動化、低單價、規模效應 |
| 按需印刷（POD） | 每次需求即時印製 | 零庫存、數位印刷主導 |
| 包裝印刷廠 | 品牌包裝、標籤 | 高精度色彩、多工序 |

---

## 印刷技術分類

| 技術 | 適合場景 | 特性 |
|------|----------|------|
| 平版印刷（Offset） | 大量商業印刷 | 色彩精準、成本低（量大時） |
| 數位印刷（Digital） | 小量 / 個人化 | 快速、無版費、適合 POD |
| 網版印刷（Screen） | 特殊材質（布料、塑料） | 油墨厚、適合白墨 |
| 凹版印刷（Gravure） | 包裝、出版大量印刷 | 超高品質、版費昂貴 |
| 數位噴墨（Inkjet） | 大幅輸出、標牌 | 尺寸彈性大 |

## 產業痛點（關鍵背景）

### 客戶端痛點
- **稿件品質不穩定**：客戶使用非專業軟體（如 Word、Canva），常出現解析度不足、RGB 色彩、缺乏出血等問題，導致退稿率高。
- **改稿成本高**：溝通成本大，來回確認拖慢交期。
- **版本混亂**：多次修稿後版本管理混亂，印錯版本。

### 印廠端痛點
- **排程複雜**：多工單、多機台、多工序，排程最佳化困難。
- **交期壓力**：急單插入打亂計畫排程。
- **物料管理**：紙張、油墨庫存預測困難，易造成缺料或積料。
- **色彩一致性**：不同批次印刷色差問題，需精細的色彩管理。
- **人工作業多**：多為手動操作，數位化程度低。

## 數位化轉型趨勢

| 趨勢 | 說明 | 對 PM 的影響 |
|------|------|--------------|
| Web-to-Print | 客戶線上自助設計下單 | 圖編輯器的核心場景 |
| 自動化 Preflight | 自動檢查稿件規格 | 降低人工退稿率 |
| ERP 整合 | 訂單管理連結生產排程 | ERP 產品的核心價值 |
| 色彩標準化 | ICC profile 自動套用 | 品質提升關鍵 |
| 按需生產 / POD | 數位印刷驅動小批量生產 | 改變庫存與排程邏輯 |
| AI 設計輔助 | 自動版面、智能建議 | 圖編未來功能方向 |

## 關鍵法規與標準

| 標準 | 說明 |
|------|------|
| ISO 12647 | 印刷色彩標準（Offset 印刷） |
| PDF/X 系列 | 印刷用 PDF 規格（PDF/X-1a、PDF/X-4 等） |
| GRACoL / SWOP | 美規印刷色彩標準 |
| Fogra | 歐規印刷色彩標準 |

## 產品定位背景

### ERP 系統
- 目標客群：中型印刷廠（50–500 人）
- 痛點核心：排程、物料、訂單管理三大模組
- 競爭對手：傳統 ERP 廠商（如 SAP、鼎新）+ 印刷專用系統
- 差異化：印刷業深度客製 + 現代 UX + SaaS 彈性

### 線上圖編輯器
- 目標客群：有 Web-to-Print 需求的印刷廠 / 品牌客戶
- 痛點核心：讓非設計師的終端客戶能自助完成符合印刷規格的設計
- 競爭對手：Canva、Adobe Express + 印刷廠自研工具
- 差異化：印刷規格內建（出血、CMYK、Preflight）+ 白標部署

## 常見印刷廠組織架構

| 部門 | 職責 |
|------|------|
| 業務部 | 接單、報價、客戶關係 |
| 製前部（PrePress） | 稿件處理、拼版、版材製作 |
| 生產部 | 印刷機台操作 |
| 後加工部 | 裁切、燙金、模切、裝訂 |
| 倉儲物流 | 紙張採購、成品出貨 |
| 品管部 | 色彩校正、成品品質檢驗 |
