# 印刷業共用術語表（完整版）

跨 ERP 與圖編輯器產品共用的術語、縮寫與行業語言。

---

## 印刷流程術語

| 術語 | 英文 / 縮寫 | 定義 |
|------|-------------|------|
| 完稿 | Print-ready file | 可直接送印的設計稿，規格需符合印廠要求 |
| 出血 | Bleed | 印刷稿延伸超出裁切線的安全邊界，通常 3mm |
| 裁切線 | Trim line | 印後裁切的最終尺寸邊界 |
| 安全線 | Safety margin | 距裁切線內縮的設計安全區（通常 3mm） |
| 折線 | Fold line | 折頁位置標記 |
| 打樣 | Proof / Proofing | 印刷前製作的樣稿，供客戶確認 |
| 數位打樣 | Digital proof | 以噴墨或螢幕模擬最終印刷色彩效果 |
| 實體打樣 | Hard proof | 用正式印刷機印製的實物樣稿 |
| 拼版 | Imposition | 將多份稿件排列於同一印張以提升效率 |
| 印張 | Sheet | 一次送入印刷機的紙張單位 |
| 開數 | Paper size format | 印張裁切份數，如全開、對開、四開、八開 |

## 色彩管理術語

| 術語 | 英文 / 縮寫 | 定義 |
|------|-------------|------|
| 色彩模式 | Color mode | CMYK（印刷）或 RGB（螢幕）的色彩表示方式 |
| CMYK | Cyan Magenta Yellow Black | 印刷四色疊印色彩模式 |
| RGB | Red Green Blue | 螢幕光色彩模式，不適合直接印刷 |
| 特別色 | Spot color | 預先調製的單一油墨色彩，如 Pantone |
| Pantone / PMS | Pantone Matching System | 全球通用的特別色標準色號系統 |
| 色差 | Delta E / ΔE | 兩色之間的可見色彩偏移量，數值越小越準確 |
| 色彩校正 | Color calibration | 校準輸出設備以確保色彩一致性 |
| ICC profile | — | 色彩設備特性描述檔，確保跨設備色彩一致 |
| 色域 | Color gamut | 特定設備或色彩模式可呈現的色彩範圍 |
| 富墨量 | Total ink coverage / TIC | CMYK 四色加總百分比，通常上限為 300%–340% |

## 印刷技術術語

| 術語 | 英文 / 縮寫 | 定義 |
|------|-------------|------|
| 網點 | Halftone dot | 印刷用來模擬連續色調的細小點陣結構 |
| 網線數 | LPI（Lines Per Inch） | 每英吋的網線密度，越高細節越豐富 |
| 分辨率 | DPI（Dots Per Inch） | 每英吋像素點數，印刷通常需 300 dpi 以上 |
| 套印 | Overprint | 多色油墨疊印，不挖空底層顏色 |
| 鏤空 | Knockout | 底層顏色挖空，由上層顏色填滿 |
| 疊印黑 | Rich black | CMYK 四色混合的深黑，比單色黑更飽和 |
| 模切 | Die-cutting | 依設計輪廓裁切紙張的加工工序 |
| 上光 | Coating / Varnish | 表面保護塗層，分霧光、亮光、局部上光 |
| 覆膜 | Lamination | 在印品表面貼合保護膜（霧面 / 亮面） |
| 燙金 | Foil stamping | 將金屬箔燙壓於紙張表面的裝飾工序 |
| 軋型 | Emboss / Deboss | 凸壓 / 凹壓製造立體感的加工工序 |
| 白墨 | White ink | 在深色或透明材質上印製白色底層 |

## 紙張與材料術語

| 術語 | 英文 / 縮寫 | 定義 |
|------|-------------|------|
| 磅數 | GSM（g/m²） | 紙張重量，每平方公尺克重 |
| 銅版紙 | Coated paper | 表面塗布的光滑紙，適合高品質印刷 |
| 道林紙 | Uncoated paper | 無塗布的紙，手感較粗，適合書寫 |
| 特種紙 | Specialty paper | 含特殊質感或功能的非標準紙材 |
| 卡紙 | Card stock | 較厚的紙板，用於名片、包裝等 |

---

## 客戶流程術語

| 術語 | 英文 | 定義 |
|------|------|------|
| 稿件 | Artwork / File | 客戶提供的設計檔案 |
| 確稿 | File approval | 客戶確認稿件正確可送印 |
| 修稿 | Revision | 對設計稿進行修改調整 |
| 退稿 | File rejection | 印廠因規格不符退回稿件 |
| 檔案檢查 | Preflight | 自動化檢查稿件是否符合印刷規格 |
| 交期 | Delivery date | 訂單完成並交付的預定日期 |
| 急單 | Rush order | 需要優先排程加急處理的訂單 |
