# 線上圖編輯器術語表（完整版）

線上圖編輯器（B2B SaaS 設計工具）專用術語，涵蓋編輯器功能、印刷規格整合、平台架構三大面向。

---

## 核心編輯器功能術語

| 術語 | 英文 | 定義 |
|------|------|------|
| 模板 | Template | 預設設計稿，限定可編輯區域供客戶修改 |
| 可編輯區域 | Editable zone | 模板中允許客戶自行修改的圖文區塊 |
| 鎖定元素 | Locked element | 模板中禁止客戶修改的設計元素（品牌色、LOGO 等） |
| 畫布 | Canvas | 編輯器的設計工作區域 |
| 圖層 | Layer | 設計元素的堆疊順序，可調整前後關係 |
| 素材庫 | Asset library | 提供給客戶使用的圖片、圖示、圖形資源庫 |
| 字型庫 | Font library | 可用字型集合，含授權控管 |
| 文字框 | Text box | 可輸入與格式化文字的元素容器 |
| 佔位符 | Placeholder | 模板中標示「請填入內容」的位置元素 |
| 參考線 | Guide | 協助對齊的視覺輔助線（不會印出） |
| 標尺 | Ruler | 量測尺寸的工具列 |
| 縮放 | Zoom | 放大 / 縮小畫布的檢視操作 |
| 復原 / 重做 | Undo / Redo | 撤銷或恢復上一步操作 |
| 對齊工具 | Alignment tools | 元素自動對齊、等距分佈的功能 |
| 群組 | Group | 將多個元素合併為一個可整體操作的群組 |
| 複製貼上 | Copy / Paste | 元素的複製與貼上 |

---

## 印刷規格整合術語

| 術語 | 英文 | 定義 |
|------|------|------|
| 出血線 | Bleed line | 編輯器顯示的出血邊界（通常 3mm） |
| 安全線 | Safety line | 建議設計元素保持在此線內的邊界 |
| 裁切線 | Trim line / Cut line | 最終成品尺寸邊界線 |
| 解析度警告 | Resolution warning | 當圖片解析度不足 300dpi 時的警示提示 |
| 色彩模式檢查 | Color mode check | 確認設計使用 CMYK（非 RGB）的驗證機制 |
| Preflight 檢查 | Preflight | 輸出前自動檢查所有印刷規格是否合規 |
| 白墨圖層 | White ink layer | 為深色或透明材質印製的白色底色圖層 |
| 富墨警告 | Ink density warning | 當 CMYK 總墨量超過上限時的提示 |
| PDF 輸出 | PDF export | 將完稿匯出為印刷用 PDF（通常為 PDF/X-4） |
| 完稿下載 | Print-ready download | 客戶完成設計後下載符合印刷規格的完稿 |

---

## 模板管理術語

| 術語 | 英文 | 定義 |
|------|------|------|
| 模板分類 | Template category | 按產品類型（名片、DM、包裝等）分類的模板群組 |
| 模板建立 | Template creation | 設計師在後台建立可供客戶使用的模板 |
| 模板版本 | Template version | 模板的迭代歷史，支援版本回溯 |
| 模板發佈 | Template publish | 將模板從草稿狀態切換為客戶可見的公開狀態 |
| 私有模板 | Private template | 僅限特定客戶或企業可見的模板 |
| 公共模板 | Public template | 所有客戶均可使用的通用模板 |
| 客製模板 | Custom template | 依特定客戶需求專屬建立的模板 |
| 模板尺寸 | Template size | 成品完稿尺寸，含出血的完整尺寸 |

---

## 協作功能術語

| 術語 | 英文 | 定義 |
|------|------|------|
| 協作編輯 | Collaborative editing | 多位使用者同時在同一設計稿上編輯 |
| 即時同步 | Real-time sync | 多人編輯時的即時畫面同步更新 |
| 評論 | Comment | 在設計稿特定位置留下文字評論 |
| 分享連結 | Share link | 產生可分享的設計稿預覽或編輯連結 |
| 審核流程 | Approval workflow | 設計稿的多階段審核 / 確稿流程 |
| 版本歷史 | Version history | 設計稿各時間點的自動儲存紀錄 |

---

## 平台架構術語

| 術語 | 英文 | 定義 |
|------|------|------|
| 白標部署 | White-label | 印刷廠以自有品牌形象部署圖編輯器給其客戶使用 |
| 租戶 | Tenant | 使用白標圖編平台的印刷廠（每家為一個租戶） |
| 工作區 | Workspace | 單一租戶（印刷廠）內的設計管理空間 |
| 角色權限 | Role & Permission | 設計師、管理員、客戶等不同角色的操作權限 |
| API 串接 | API Integration | 圖編輯器與 ERP、電商平台的資料交換介面 |
| 嵌入式編輯器 | Embedded editor | 以 iframe 或 SDK 嵌入印刷廠自有網站的編輯器 |
| SSO | Single Sign-On | 讓用戶以印刷廠帳號系統直接登入圖編輯器 |
| CDN | Content Delivery Network | 素材庫圖片的全球加速分發網路 |

---

## 使用者體驗術語

| 術語 | 英文 | 定義 |
|------|------|------|
| 拖放 | Drag & Drop | 以滑鼠拖曳方式操作元素或素材 |
| 即時預覽 | Live preview | 設計變更後即時渲染顯示結果 |
| 3D 預覽 | 3D mockup preview | 模擬設計成品於實體物品上的效果（如杯子、包裝盒） |
| 響應式設計 | Responsive design | 編輯器界面適應不同螢幕尺寸的能力 |
| 鍵盤快捷鍵 | Keyboard shortcuts | 常用操作的鍵盤快捷鍵支援 |
| 自動儲存 | Auto-save | 設計過程中的自動定期儲存 |
| 空白狀態 | Empty state | 工作區無內容時的引導介面設計 |
| 載入狀態 | Loading state | 操作進行中的視覺等待提示 |

---

## 常見使用者角色

| 角色 | 說明 |
|------|------|
| 終端客戶 | 使用圖編輯器進行自助設計的印刷採購者（非設計師） |
| 設計師 | 負責建立模板、維護素材庫的專業設計人員 |
| 平台管理員 | 管理租戶設定、用戶權限、模板發佈的管理角色 |
| 系統整合者 | 負責 API 串接的技術人員 |
